Object Oriented Design:
GUI and Console Calendar

Designed and created a console calendar application in java. User can create and add an event; can toggle view of the calendar between a day view and a month view; can go to a date, among other features. At the end, it saves the events to a text file which can be loaded again. Here is a link to the repository where I have my source files. I later improved this project by adding a GUI component. The Component displays the month view, which each day clickable. It also has the day view of events. As well as previous and next buttons.
